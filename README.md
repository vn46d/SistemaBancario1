Sistema Bancario
Projeto desenvolvido em Java com o objetivo de praticar lógica de programação, entrada de dados pelo teclado e estruturas de decisão.

Funcionalidades
Conferir saldo
Sistema de Deposito e saque 
Validação de operação inválida

Tecnologias Utilizadas
Java
IntelliJ IDEA
Git
GitHub
Como Executar
Clone o repositório:
git clone https://github.com/vn46d/SistemaBancario1
Abra o projeto no IntelliJ IDEA.

Execute a classe Bancos

Digite os valores e a operação desejada no terminal.

Exemplo de Uso
======= Sistema Do Vinicius ========

Digite o valor de deposito: 2 
10

Solcite o Saldo: 1
10

Digite uma das caracteres: Adicionar,1,2,3
